If you want the Web Blades WeakAura to play a sound when it's safe to move, make the following modifications. The images are what each tab should look like.

1. Inder the trigger tab find "Custom Variables" and add copy and paste the following:

  ```lua
  {
    expirationTime = true,
  }
  ```

2.   Then under the custom options tab add a condition where `Remaining Duration = 2.5`, then `Sound â†’ Play`, pick your sound of choice, then use the `Master` sound channel.
